MeasureIt

UX • UI

MeasureIt is an simple iOS application designed to make home renovations and decorating easier. For my first UI project, the goal was to create a simple iOS utility app that served a simple but helpful function. Tools are always handy, but we aren’t always prepared for what the day throws at us. Using the GPS capabilities, the app would allow the user to be able to use their phone to measure and log their data. With the option to store measurements in list form or simply applying data on a photo of a room, every user has the tools that suits their needs.
